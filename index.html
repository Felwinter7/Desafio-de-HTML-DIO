Cliente x Server

voltar

Quando utilizamos a internet temos uma ação chamada de interação de cliente x servidor, esta interação ocorre quando utilizamos a internet para acessar um determinado conteúdo. Quando acessamos um site na internet estamos abrindo um website forncendo o conteúdo com dados armazenados em um servidor, ao clicarmos no link do website estamos solicitando o acesso para estes determinados conteúdos e o tempo de espera que também é conhecido por loading é o intervalo de tempo para o servidor receber a solicitação de acesso aos dados e responder enviando ao cliente o que o mesmo solicitou.

O servidor se trata de um ou mais computadores que serve para armazenar dados como documentos, imagens, audios e etc. Podendo tambem ser utilizado para hospedar programas que respondem as solicitaçoes dos clientes e processando elas. No servidor é processado todas as aplicações de determinadas linguagens de programação como: Java, C#, PHP e etc.

O que é o HTML?

voltar

Ao iniciar os estudos programação, muitas pessoas confundem o HTML como uma das linguagens de programação, porém o HTML se trata de uma Linguagem de Marcação de Hipertexto(HiperText Markup Language) e ele permite inserir o conteudo e fazer a estrutura de um website. Com o HTML nós criamos aquilo que observamos ao acessar um site e com ele podemos estrurura adicionando funções que podem facilitar a utilização do site para quem estiver utilizando.

O uso das Tags no HTML

voltar

As Tags são códigos que informam ao navegador a estrutura do site ao interpretar o HTML assim permitindo o criador do website que crie o modelo e formato da estrutura, lembrando que sempre que utilizarmos as Tags, devemos colocar <> (sinal de abertura de uma tag) e selecionamos a tag que deseja utilizar, colocamos o conteudo e terminamos utililizando o (sinal de término da tag) para fechar a tag e definir a função que a tag deve aplicar nos dados escolhidos entre os sinais de inicio e término. Quando utilizarmos as tags podemos: utilizar a tag h para definir o titulo dos conteudos inseridos no site podendo ir do "h1" ao "h6", para escrevermos um paragrafo devemos utilizar a função "p" que nela iremos inserir os textos adicionados em nosso sites, existem tags que podem personalizar o tamannho das palavras como a "small", temos tags que podemos colocar as letras acima ou abaixo em relação a linha da escritas como o sup que deixa o conteúdo na parte superior e o sub que coloca o conteúdo na pagina inferior. E lembre-se ao utilizar as tags nunca esqueçam de utilizar "<>" ao utilizar as tags, pois uma vez que queremos aplicar determinada tag sem informar o sinal de abertura a mesma não vai ser aplicada ex: "body" "<"body">"

Caso queira explorar mais tags novas e entender suas funções, modo de uso e como aplicar em seu HTML podemos utilizar o dicionário acessando o site https://www.w3schools.com/tags/ .

  1. A História do HTML

    2. O HTML foi inventado por Tim Berners-Lee, um físico do centro de pesquisas CERN, na Suíça. Ele surgiu com a ideia de um sistema de hipertexto na internet. Hipertexto significa um texto que possui referências (links) para outros textos que podem ser acessados imediatamente. Ele publicou a primeira versão do HTML em 1991, consistindo em 18 tags. Desde então, cada versão do HTML vem com novas tags e atributos (modificadores de tags). Devido a rápida ascensão e popularidade, o HTML é agora considerado um padrão oficial da web. O maior upgrade da linguagem foi o lançamento do HTML5 em 2014. Diversas novas tags semânticas foram adicionadas que revelam o significado do seu próprio conteúdo, como "article", "header", e "footer".
